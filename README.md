# My-Digest-Journal-App
Personal journal web app that analyzes and summarizes user's entries

### For the initial setup
- Clone git repo
- CD to repo directory
- >>> python3 install -r requirements.txt
- Create .env file
- Set test user credentials in the .env file  
    TEST_EMAIL=''    
    TEST_PASSWORD=''    
    TEST_NAME=''    

### To run app
- CD to repo directory
- >>> python3 app.py
- >>> Open http://127.0.0.1:5000 in browser
